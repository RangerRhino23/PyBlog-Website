Using Python and Sockets to Create a Chatroom.

Difficulty: Intermediate

Introduction


In this blog we will be covering how to make a chatroom using python and its built-in module "socket".

First off you'll want to import the module into the python script. But first double-check you have it installed by running "pip install socket" then you can import it 


import socket


Using this import statement you have now successfully import the socket module and we can continue onto the next step of creating a chatroom with python.
