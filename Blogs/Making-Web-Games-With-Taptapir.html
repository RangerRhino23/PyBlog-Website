Making Web Games with Tapair.


Difficulty: Beginner

Introduction

There are many different ways to create a working game on the web. You could use a website like Scratch that uses code blocks to create the game or you could code one using text coding like JavaScript. Although there are many different ways, Taptapir is in my opinion one of the easier ones to learn and has tons of built in features that make creating games so much easier.

In this blog post, we will take a closer look at Taptapir and explore some of its features. We will also provide some examples of Taptapir code and discuss its potential applications.

What is Taptapir?

Taptapir is a new programming language created by Pokepetter. It is a language that runs in the browser and transpiles into JavaScript. Taptapir is a statically typed language with indentation-sensitive syntax, meaning that the indentation of your code determines the structure of your program. This makes Taptapir code more readable and easier to write. Taptapir is also significantly faster than Python, making it a good choice for performance-critical applications.

The super useful thing about Taptapir is that its coded in a syntax like Python. It uses syntax like Ursina which is a game engine for Python also created by Pokepetter.

Getting Started

To get started with Taptapir, first create a project folder. This is where all the code for your game will be stored. Inside of your project folder, create a new file called game.html. Once you have that file created, open that folder in your IDE or editor of choice.

Installing required Taptapir Files

Unlike Urisna and Python, Taptapir can't be installed using pip. To install Taptapir, you will have to download its required sorce code from github. To do this, open a command terminal in your project folder. In the terminal, run the following command: 


git clone https://github.com/pokepetter/taptapir.git --depth=1


This should create a folder inside of your project folder called "taptapir".

Creating a blank scene

Now that you've installed taptapir, you can start to create your web game. In the html file you created earlier, put the following code: 


<html><title>title</title><body>
<script src="taptapir/taptapir.js"></script>
<script type="text/sunsnake">
# code here
</script><script src="taptapir/sunsnake_compiler.js"></script></body></html>


This creates a blank green window that you will be able to put different Entities and Buttons into. The creator, Pokepetter has created a documentation page that you can look through to help you. you can access that page here.

Creating a simple clicker

A simple starter project to create in Taptapir is a clicker. These are simple but also fun to make. Try making it yourself but if your stuck at all, feel free to use code from this example: 


<html><title>title</title><body>
<script src="taptapir/taptapir.js"></script>
<script type="text/sunsnake">
set_background_color('#fff')
set_window_color('#fff')
score = 0
text = Text(text='00', text_origin=[0,0], scale=1, y=.2, text_size=8)
b = Button(text='Click me!', color=hsv(160,1,.6), scale=[.3,.15], text_color=color.white)
b.on_click = def():
    score += 1
    text.text = `${score}`
    b.scale = [e*1.25 for e in [.3,.15]]
    b.animate('scale_x', .3, duration=.1)
    b.animate('scale_y', .15, duration=.1)
</script><script src="taptapir/sunsnake_compiler.js"></script></body></html>


Conclusion

Taptapir is a very simple way to create web games while using Python syntax. If you want to see more Taptapir projects, search for them at the top of the screen!
