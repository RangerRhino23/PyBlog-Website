Making Web Games with Tapair.


Difficulty: Beginner

Introduction

There are many different ways to create a working game on the web. You could use a website like Scratch that uses code blocks to create the game or you could code one using text coding like JavaScript. Although there are many different ways, Taptapir is in my opinion one of the easier ones to learn and has tons of built in features that make creating games so much easier.

In this blog post, we will take a closer look at Taptapir and explore some of its features. We will also provide some examples of Taptapir code and discuss its potential applications.

What is Taptapir?

Taptapir is a new programming language created by Pokepetter. It is a compiled language that runs in the browser and transpiles into JavaScript. Taptapir is a statically typed language with indentation-sensitive syntax, meaning that the indentation of your code determines the structure of your program. This makes Taptapir code more readable and easier to write. Taptapir is also significantly faster than Python, making it a good choice for performance-critical applications.

The super useful thing about Taptapir is that its coded in a syntax like Python. It uses syntax like Ursina which is a game engine for Python also created by Pokepetter.

Getting Started

To get started with Taptapir, first create a project folder. This is where all the code for your game will be stored. Inside of your project folder, create a new file called game.html. Once you have that file created, open that folder in your IDE or editor of choice.

Installing required Taptapir Files

Unlike Urisna and Python, Taptapir can't be installed using pip. To install Taptapir, you will have to download its required sorce code from github. To do this, open a command terminal in your project folder. In the terminal, run the following command: 


git clone https://github.com/pokepetter/taptapir.git --depth=1


This should create a folder inside of your project folder called "taptapir".
