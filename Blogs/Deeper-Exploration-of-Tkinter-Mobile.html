A Deeper Exploration of Tkinter

Difficulty: Beginner

Quick recap.

From what we saw in this blog you would've created a label, a button and a function for the button when clicked.
In this blog we go over so better functions for placement.

Using .grid

Using grid can allow you to more accuractly place buttons and labels. Using the grid function is fairly simple and takes only 2 required args (arguments) which are "column" and "row" which you can infer what they do. 

import tkinter as tk

def button_click():
    print("Button clicked!")

root = tk.Tk()

label = tk.Label(root, text="Hello, world!")
label.grid(column=0,row=0)

button = tk.Button(root,text="Click me!", command=button_click)
button.grid(column=0,row=1)

root.mainloop()


this code will do the exact same as the code which is in the previous blog but uses a grid system which can make putting buttons and labels, like aforementioned, more accuractly

Changing Label Text

You can change the text on a label using the .config function, this can also allow you to change the column and or row on which the label is placed on 


def button_click():
    label.config(text="You've changed me!")


Here we have modified the "button_click()" function so when the button is clicked its changes the text of the label to "You've changed me".

What we've covered:

Function Functionality
.row Can be used to place labels on a grid
.config Can be used to change things like the text of a label

In Conclusion:

We've learnt quite a few essential functions that are apart of tkinter, using grid and config to change the position and text of a label and now you have a better understanding of how to create applications with tkinter.
