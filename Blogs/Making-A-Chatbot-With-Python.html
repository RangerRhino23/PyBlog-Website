Chatbots: How to make them in python

Difficulty: Intermediate

What is a chatbot?

A chatbot is an artificial intelligence (AI) program designed to interact with users through conversation. It can simulate human-like conversations and provide automated responses based on predefined rules or machine learning algorithms.

Creating a chatbot with OpenAI in Python

To create a chatbot using OpenAI in Python, we'll be using the OpenAI API, specifically the GPT-3.5 model. Follow the steps below:


  • Install the OpenAI Python package by running the following command in your terminal:

    • pip install openai


  • Import the OpenAI library in your Python script:

    • import openai


  • Set up your OpenAI API credentials by providing your API key:

    • openai.api_key = 'YOUR_API_KEY'


    You can find your openAI key here it should look like this:


    You may have to create a new API key which you can do by clicking "Create new secret key"


  • Define a function to interact with the chatbot. Here's a simple example:

    • def chat_with_bot(message):
    response = openai.Completion.create(
        engine='text-davinci-003',
        prompt=message,
        max_tokens=50,
        temperature=0.7,
        n=1,
        stop=None,
    )
    return response.choices[0].text.strip()


  • Call the chatbot function and provide a user message to get a response:

    • while True:
    user_message = input("Enter input: ")
    bot_response = chat_with_bot(user_message)
    print(bot_response)

    What the code should look like
