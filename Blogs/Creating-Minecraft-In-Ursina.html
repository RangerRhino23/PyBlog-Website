Creating A Minecraft Clone In Ursina

Difficulty: Intermediate

Edited: 06/01/2023

Introduction

Creating games in Ursina is very simple and fun. While you can go creating simple 2D games like a Cookie Clicker or Platformer game, creating 3D Games are much more fun but also more challenging. In this blog we will create a Minecraft Clone and learn more about using Ursina to create games!

Setting Up Project Folder

The first thing when creating any Ursina game is creating a project folder. In this blog I am assuming you have already installed Python and have a decent knowledge about Python. If you don't click here for a Python Installation guide and here for a Python Introduction. To create a project folder, navigate to the place you store all of your project files. Create a new folder and name it what ever you want. I would recommend naming it "Minecraft-Clone" or something close to that. After you create your project folder, open that folder in your desired editor or IDE.

Importing required modules

To create a Ursina game, you will need to import ursina. If you are wanting to create a first person 3D game, you will need to install ursina's first person controller. To import the required stuff and creating a Ursina window, write the following code in a new python file: 



from ursina import *
from ursina.prefabs.first_person_controller import FirstPesronController

app = Ursina()

Creating a Voxel Class

When creating a Minecraft Clone, you will need to create multiple blocks that you can interact with. Instead of creating a block with all the required properties, you can create a class that defines all of these qualities in a preset that you can call when ever you want to create a new block or voxel. In the python file, add the following code: 



class Voxel(Button):
def __init__(self, position=(0,0,0)):
    super().__init__(parent=scene,
        position=position,
        model='cube',
        origin_y=.5,
        texture='white_cube',
        color=color.color(0, 0, random.uniform(.9, 1.0)),
        highlight_color=color.lime,
    )

Creating the environment

After creating a voxel class, you can create the envrionment that the player can interact with. In the python file, add the following code: 



for z in range(8):
    for x in range(8):
        voxel = Voxel(position=(x,0,z))

Creating the input function

After creating the voxel class and creating the environment, we need to write the code that allows the player to interact with the environment; Creating/Destroying the environment. In the file, add the following code: 



def input(key):
    if key == 'left mouse down':
        hit_info = raycast(camera.world_position, camera.forward, distance=5)
        if hit_info.hit:
            Voxel(position=hit_info.entity.position + hit_info.normal)
    if key == 'right mouse down' and mouse.hovered_entity:
        destroy(mouse.hovered_entity)

        application.quit()

Thats it! That is all the code required to create a Minecraft Clone in Ursina!
